# Clinic Management System

A console-based clinic management system built in Java that handles patient care workflows, doctor management, pharmacy operations, and medical reporting using custom data structures.

## Features
- Patient and doctor management with CRUD operations
- Consultation scheduling and tracking
- Treatment and prescription management
- Pharmacy inventory and transaction handling
- Custom Set and Queue ADT implementations
- Medical reporting and data analytics
- Input validation and data initialization with sample data (220+ treatments and prescriptions)

## Installation
```bash
# clone the repo
git clone https://github.com/yourusername/clinic-management-system.git

# navigate to project directory
cd clinic-management-system

# compile the project using Ant
ant compile

# run the application
ant run
```
